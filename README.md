# ADM Soluções - Sistema de Gestão de Agendamentos

## 📌 Sobre o Projeto
O **ADM Soluções** é uma aplicação web cliente-servidor desenvolvida para automatizar e gerenciar o ciclo completo de agendamentos realizados por um prestador de serviços. 
O projeto foi idealizado para substituir a gestão manual e a comunicação ineficiente, fornecendo uma interface intuitiva para que os clientes consultem disponibilidade e realizem reservas de forma autônoma.

## 🚀 Funcionalidades
- Agendamento online em tempo real para consulta e reserva de horários.
- Controle total para o administrador cadastrar, editar ou cancelar serviços.
- Base de dados para manter registos de clientes e histórico de atendimentos.
- Geração de relatórios para acompanhamento financeiro do negócio.
- Sistema de avaliação para que os clientes deixem notas e comentários sobre os serviços.

## 🛠️ Tecnologias Utilizadas 
- **HTML5, CSS3, JavaScript**
- **PHP**
- **MySQL**
- **Arquitetura: Padrão em camadas**
